Syntax Of Control Flow

What is Control Flow?

Control flow refers to the order in which individual statements, instructions, or code blocks are executed in a program...

There are several fundamental control flow structures in programming:

  • Sequential Execution: Statements are executed one after the other, in the order in which they appear in the code.
  • Conditional Statements: Code branches based on a condition. Common structures include `if`, `else if`, and `else`.
  • Looping Statements: Code is repeated multiple times based on a condition. Common loop structures include `for`, `while`, and `do-while`.
  • Switch Statements: Allows a program to evaluate an expression against multiple possible case values.

Control flow structures enable developers to create dynamic and flexible programs by making decisions and executing specific blocks of code based on those decisions...

IF Syntax

The "if" statement is a fundamental control flow statement used to make decisions in code. It allows you to execute a block of code if a specified condition is true. Here's the basic syntax of an "if" statement in several programming languages: 

if (condition) {
    // Code to be executed if the condition is true
}

var age = 25;

if (age >= 18) {
    console.log("You are an adult.");
}

If Else Syntax

The if-else statement is a fundamental control flow structure in programming. It allows a program to make decisions based on whether a specified condition is true or false. Here's a general description of the if-else statement: 

if (condition) {
    // Code to be executed 
    // if the condition is true
} else {
    // Code to be executed 
    // if the condition is false
}

var age = 25;

if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

Switch Syntax

The switch statement is another control flow statement used in programming to evaluate an expression against multiple possible case values. It provides an alternative to a series of if-else if-else statements when you have a single expression with multiple possible matches. Here's the basic syntax for the switch statement in various programming languages: 

switch (expression) {
    case value1:
        // Code to be executed 
        // if expression matches value1
        break;
    case value2:
        // Code to be executed 
        // if expression matches value2
        break;
    // ... more cases
    default:
        // Code to be executed 
        // if no case matches the expression
}

var day = "Monday";

switch (day) {
    case "Monday":
        console.log("It's the first day of the week.");
        break;
    case "Friday":
        console.log("It's almost the weekend!");
        break;
    default:
        console.log("It's a regular day.");
}

For Loop Syntax

A for loop is a control flow statement that allows you to repeatedly execute a block of code for a specified number of times. The general syntax of a for loop varies slightly between programming languages, but the basic structure is similar. Here's a typical representation: 

for (initialization; condition; update) {
    // Code to be executed 
    // in each iteration
}

for (var i = 0; i < 5; i++) {
    console.log(i);
}

While Loop Syntax

A while loop is another control flow statement that repeatedly executes a block of code as long as a specified condition is true. The loop continues to execute until the condition becomes false. The basic syntax of a while loop is as follows: 

while (condition) {
    // Code to be executed as long as 
    // the condition is true
}

var i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Do While Loop Syntax

A do-while loop is a control flow statement that is similar to the while loop, but with a crucial difference: the do-while loop guarantees that the block of code inside the loop is executed at least once, even if the loop condition is initially false. The general syntax is as follows: 

do {
    // Code to be executed
} while (condition);

var i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);
